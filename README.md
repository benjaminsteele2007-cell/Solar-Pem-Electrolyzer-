# Solar-Pem-Electrolyzer-
Solar-powered PEM electrolyzer prototype — simulation, firmware, data logging, and technical report
A solar-powered proton exchange membrane (PEM) electrolyzer prototype built 4/15/26 - 8/1/26.

## Project goals
- Python simulation of electrolyzer performance using Faraday's law
- Solar input power conditioning and sensor integration
- Real-time data logging and analysis
- Low-pressure hydrogen production with sensor monitoring
- Integrated technical report for professor and research lab outreach
